# Programmieren lernen

Wir arbeiten mit [Kojo](https://www.kogics.net/kojo) — einer offenen Lernumgebung, in der man mit der Programmiersprache Scala spielerisch programmieren lernt. Kojo eignet sich für Einsteiger jeden Alters und bietet Turtle-Grafik, Animationen, Spiele, Musik und vieles mehr.

## Anleitungen

- [Kojo Installation auf Windows (ohne Admin-Rechte)](docs/Kojo_Installation_Windows_ohne_Admin.md) — Schritt-für-Schritt-Anleitung für Windows-Nutzer, die keine Administratorrechte haben.
- [Kojo Aufgabenblatt 1](docs/Kojo_Aufgaben01.md) — Einfuehrung mit ersten Aufgaben zu Turtle- und Picture-Befehlen.
- [Kojo Aufgabenblatt 2](docs/Kojo_Aufgaben02.md) — Weiterfuehrende Uebungen mit Codebeispielen zu Schildkroeten- und Zeichenbefehlen.
